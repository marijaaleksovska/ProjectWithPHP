# The Italian Guide (Web Application for Italian Food in Skopje)
- project assignment.

The Italian Guide website is designed for all gourmets and lovers of
Italian cuisine. It features three Italian restaurants in Skopje: Enrico, Gino
and Veneto. On this web application you can find information about their
location, the full menu and the specialties that they offer. The users of
this application have the opportunity at any moment to reserve a place in
any of the represented restaurants. This web application is comprised of
6 parts: the starting page, the section "About Us", the section "Specialties",
the section "Restaurants", reservation section and section for logging the
user for an existing profile along with the sections for creating a new user
profile and password change.
